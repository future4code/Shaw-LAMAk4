# labenu Music Awards

Projeto proposto pela instituição Labenu.

## Funcionalidades

1. Usuário
    - Criar usuário;
    - Fazer login.

2. Banda
    - Criar Banda;
    - Visualizar detalhes de uma banda;

3. Shows
    - Criar show;
    - Buscar todos os shows de uma data;
